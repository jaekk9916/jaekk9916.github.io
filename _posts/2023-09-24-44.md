# Windows

## Eclipse 
- Getter Setter method: `Alt` + `Shift` + `S`, then `R`

## IntelliJ
- Create file: `Ctrl` + `Alt` + `Insert` key
- Program Run: `Shift` + `F10`
- Program Debug: `Shift` + `F9`
- Generate test class: `Ctrl`+`Shift`+`T`

## Execute Jekyll local server
```bash
exec bundle jekyll serve
```
